1. 請遵守比賽規則，不要使用任何作弊手段；
2. 不要在比賽中和除了本次比賽的隊友之外的任何人交流有關題目的信息與思路，更不要分享 FLAG；
3. 請遵守社區規範，不要在比賽交流群中發布任何不友善的言論；
