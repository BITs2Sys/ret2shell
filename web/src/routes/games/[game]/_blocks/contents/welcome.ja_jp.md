ここで知識と楽しさのために戦う高品質な競技をお楽しみいただけることを願っています。参加を向上させるために、以下のガイドラインに従っていただきたいと思います：

1. コンテストのルールに従い、不正行為を避けてください。
2. コンテスト中に問題に関連する情報や解決策をチームメイト以外の誰かと話すことは避け、回答を共有しないでください。
3. コミュニティの規範を守り、コンテストのコミュニケーショングループで不友好な発言を控えてください。
4. コンテストが終了するまで、公共の場で問題の詳細や戦略について話すことを避けてください。
5. コンテストが終了し、コンテスト資料がアーカイブされるまで、完全な解答を公に共有しないでください。
6. 主催者からの特別な指示がある場合があります。コンテスト前に通知されます。主催者の判断が最終的です。

これらのガイドラインに従わない場合、失格やアカウントの停止の対象となる可能性があります。ほとんどの参加者が友好的であると信じています。あなたもそうであることを願っています！ ΦωΦ
