# Configuration Files

## `config.toml` (required)

> provided as [config.sample.toml](config.sample.toml) at this directory.

The example configuration file for ret2shell.

## `license` (required)

The license file for ret2shell.

> [!TIP]
>
> By providing private key `priv.bin` at this directory, paired with the public key `pub.bin` which is integrated in the server binary file, the license file could be generated by using `r2s-license`. Examples are as follows:
>
> - **Generate new CA.**
>
>   ```shell
>   cargo run --bin r2s-license -- init -p config/
>   ```
>
>   This would generates `priv.bin` and `pub.bin` in the `config/` directory.
>
> - **Sign new license.**
>
>   By inputting with `priv.bin` and other information, a new license file could be generated at the `config/` directory.
>
>   ```shell
>   cargo run --bin r2s-license -- new --ca ./config/priv.bin --path ./config/ --issuer '<Organization>' --website '<Domain>' --level enterprise --date 2077-01-01
>   ```

## `sensitive_word_list.txt`

> Path of this file could be changed in `config.toml`.

Words in this file would trigger the sensitive word detection mechanism in the server.
