## 🌟歡迎來到練習場！

練習場中的題目**沒有時間限制**，可以無限次重試，**提示全部解鎖**，**題解全部開放**。但你在這裡的提交不會獲得任何分數，成果記錄僅供你自己參考。

部分比賽在結束後也會歸檔到練習場，並保留自己的（不含隊友的）解題狀況。比賽歸檔後，隊伍將僅被保留在比賽排行榜中，練習場的操作不會影響比賽排行等內容。

> 儘管題解已經開放，但請**不要直接抄題**！練習場的解題情況僅你自己可見，沒有任何形式的獎勵，因此在這裡抄題是沒有意義的，請對自己負責！

## 問題回報

題目的內建問題回饋 `🔨錘子` 無法在練習場內使用，請使用其他回饋途徑提交問題。

有時你可能會遇到下面的問題：

- 解題的步驟是正確的，且得到了答案，但顯示答案錯誤；
- 環境已啟動，檢查了網路問題和連線方式，依然無法連線；
- 題目描述或題解有誤。

> 出題人也會犯錯，如有問題請自信地回報問題，🔨敲打一下出題人。
