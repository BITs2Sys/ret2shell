## 🌟欢迎来到练习场！

练习场中的题目**没有时间限制**，可以无限次重试，**提示全部解锁**，**题解全部开放**。但你在这里的提交不会获得任何分数，成果记录仅供你自己参考。

部分比赛在结束后也会归档到练习场，并保留自己的（不含队友的）解题状况。比赛归档后，队伍将仅被保留在比赛排行榜中，练习场的操作不会影响比赛排行等内容。

> 尽管题解已经开放，但请**不要直接抄题**！练习场的解题情况仅你自己可见，没有任何形式的奖励，因此在这里抄题是没有意义的，请对自己负责！

## 💬问题反馈

题目的内置问题反馈 `🔨锤子` 无法在练习场内使用，请使用其他反馈途径提交问题。

有时你可能会遇到下面的问题：

- 解题的步骤是正确的，且得到了答案，但显示答案错误；
- 环境启动了，排查了网络问题和连接方式，还是连接不上；
- 题目描述或者题解有误。

> 出题人也会犯错，如有问题请自信地反馈问题，🔨锤一下出题人。
